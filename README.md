# Netflix Data Analysis

## Project Overview
This project analyzes Netflix content using Python, Pandas, and Matplotlib to uncover trends, patterns, and insights from the Netflix dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code
- Git & GitHub

## Dataset
The dataset contains information about Netflix movies and TV shows, including:
- Title
- Type
- Country
- Release Year
- Rating
- Genre

## Key Findings
1. Movies outnumber TV Shows in the Netflix library.
2. Netflix content additions increased rapidly after 2015.
3. International Movies and Dramas are the most common genres.
4. TV-MA is the most frequent rating category.
5. The United States contributes the largest share of content.

## Visualizations
The project includes:
- Content Growth Analysis
- Movies vs TV Shows Comparison
- Rating Distribution Analysis
- Top Countries Analysis
- Top Genres Analysis
